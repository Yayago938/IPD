# SecureWipe Verification Portal

Frontend-only enterprise UI for certificate upload, hash and signature verification, wipe result dashboards, audit logs, and reports.

## Stack

- Vite
- React functional components
- Tailwind CSS
- React Router DOM
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

This project contains no backend, API calls, authentication, database integration, or real cryptographic logic. Verification states and reports are represented with mock frontend data for UI and workflow demonstration.
